class register.Register(model, metric, sampler)¶

A registration class for estimating the deformation model parameters that best solve:

f( W(I;p), T )

where:
f : is a similarity metric. W(x;p): is a deformation model (defined by the parameter set p). I : is an input image (to be deformed). T : is a template (which is a deformed version of the input).
